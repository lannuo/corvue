## 快速开始

### 构建项目
```bash
cd corvus
cargo build --release
```

### 运行示例
```bash
# 基本工作流示例
cargo run --example basic_workflow

# MCP 服务器示例
cargo run --example simple_mcp_server
```

### 使用 CLI
```bash
# 查看帮助
cargo run -- --help

# 交互式聊天
cargo run -- chat

# 运行单任务
cargo run -- run "Your task here"

# 查看配置
cargo run -- config
```
